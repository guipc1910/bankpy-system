# bankpy

🏦 BankPy — Sistema Bancário em Python

O BankPy é um projeto simples e didático desenvolvido para praticar conceitos essenciais de programação em Python. Ele simula um sistema bancário básico, permitindo operações como criação de contas, depósitos, saques e visualização de saldo.

🎯 Objetivo do projeto

Este projeto foi criado para:

Aprender e aplicar lógica de programação

Treinar o uso de funções, módulos e estruturas de dados

Entender conceitos de fluxo de caixa, validação de dados e organização de código

Construir um sistema útil como parte do meu portfólio

🔧 Funcionalidades

Criar conta bancária

Depositar valores

Realizar saques com validação

Verificar saldo

Extrato simples (estou trabalhando nisso)

Estrutura modular para facilitar melhorias

🚀 Próximos passos

Criar interface gráfica ou API

Adicionar banco de dados
